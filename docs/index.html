Zásady ochrany osobných údajov

Posledná aktualizácia: 30. október 2025

Faktúry Manager je GPT asistent pre analýzu a správu faktúr. Vaše súkromie je pre nás dôležité.

1. Aké údaje zbierame

Faktúry Manager pristupuje len k údajom vo vašom Google Sheets dokumente, ktorý ste poskytli:

  • Čísla faktúr
  • Názvy partnerov (firiem)
  • IČO a DIČ
  • Sumy faktúr a dátumy
  • Informácie o platbách
  • Meno vystavovateľa faktúry

2. Ako používame vaše údaje

Vaše údaje používame výhradne na poskytovanie služby:

  • Zobrazenie faktúr a analýz
  • Vytvorenie štatistických reportov
  • Vyhľadávanie v faktúrach
  • Analýza trendov a platobnej disciplíny

3. Ukladanie a zdieľanie údajov

Faktúry Manager NEUKLADÁ žiadne údaje trvale.

Všetky údaje zostávajú vo vašom Google Sheets dokumente. API len číta dáta na požiadanie a okamžite ich spracováva.

  • ❌ Neuchovávame kópie vašich faktúr
  • ❌ Nezdieľame vaše údaje s tretími stranami
  • ❌ Nepredávame vaše údaje
  • ✅ Údaje zostávajú pod vaším vlastníctvom a kontrolou

4. Bezpečnosť

Zabezpečenie vašich údajov:

  • API Key autentifikácia: Prístup len s autorizovaným kľúčom
  • HTTPS šifrovanie: Všetka komunikácia je šifrovaná
  • Google Service Account: Bezpečné prihlas ovanie k Google Sheets
  • Žiadne záznamy: Nelogujeme obsah vašich faktúr

5. OpenAI ChatGPT

Faktúry Manager používa OpenAI GPT na spracovanie prirodzeného jazyka. OpenAI môže spracovať vaše konverzácie podľa svojich zásad ochrany osobných údajov.

Odporúčame: Nepoužívajte citlivé osobné údaje vo vašich otázkach (napr. heslá, čísla kariet).

6. Vaše práva

Máte právo:

  • Prístup: Vaše údaje sú vždy dostupné vo vašom Google Sheets
  • Úprava: Môžete kedykoľvek upraviť údaje priamo v Google Sheets
  • Vymazanie: Môžete odstrániť údaje alebo zablokovať prístup API
  • Odvolanie súhlasu: Môžete kedykoľvek prestať používať Faktúry Manager

7. Tretie strany

Faktúry Manager používa tieto služby:

  • Google Sheets API: Na prístup k vašim údajom
  • Railway: Na hosting API servera
  • OpenAI ChatGPT: Na spracovanie prirodzeného jazyka

8. Cookies

Faktúry Manager nepoužíva cookies. API server nepracuje s cookies ani trackingom.

9. Zmeny zásad

Tieto zásady môžeme aktualizovať. Dátum poslednej aktualizácie nájdete na začiatku tohto dokumentu.
